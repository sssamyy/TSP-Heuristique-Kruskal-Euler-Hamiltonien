# TSP-Heuristique-Kruskal-Euler-Hamiltonien
Implémentation d'une heuristique pour le problème du voyageur de commerce (TSP) avec génération de matrices aléatoires, MST par Kruskal, doublage des arêtes, parcours eulérien (Hierholzer), puis transformation en cycle hamiltonien. Analyse de complexité et visualisation des graphes.
